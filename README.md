# Analyse RGPD des données Instagram – Impact comportemental de l’utilisateur
## Présentation du projet

Ce projet a pour objectif d’analyser les données personnelles Instagram obtenues via la procédure d’export RGPD.
L’analyse vise à étudier l’impact d’Instagram sur le comportement de l’utilisateur, en distinguant : L’usage passif (consommation de contenu), L’usage actif (interactions) et les comportements globaux et tendances d’utilisation

L’outil développé est conçu pour être générique, automatisé et respectueux de la vie privée.

RGPD & Éthique: Toutes les données utilisées dans ce projet proviennent exclusivement du compte Instagram de l’utilisateur, via la fonctionnalité officielle d’export RGPD. Aucune donnée personnelle n’est partagée, publiée ou stockée dans ce dépôt GitHub.

Les fichiers sensibles sont exclus grâce au fichier .gitignore.

Technologies utilisées Python 3:

Traitement de fichiers JSON et CSV

Analyse de données avec les bibliothèques standard

Visualisation de données optionnelle avec Matplotlib

Analyses réalisées

1️⃣ Analyse de l’utilisateur passif

- Habitudes de consommation de contenu

- Fréquence et évolution temporelle

- Contenus les plus consultés ou appréciés

2️⃣ Analyse de l’utilisateur actif

- Likes, commentaires, messages, stories

- Fréquence d’engagement

- Évolution de l’activité dans le temps

3️⃣ Analyse comportementale globale

- Classification du profil utilisateur

- Calcul du score d’engagement

- Évaluation du niveau de visibilité

- Interprétation du comportement numérique


 Limites du projet

- Absence d’accès aux données en temps réel

- Impossibilité de mesurer le temps réel passé sur l’application

- Interprétation comportementale partiellement subjective

- Seuils d’analyse définis de manière empirique


 Pistes d’amélioration

- Créer une interface pour que ça soit plus pratique

- Intégration avec des API officielles

- Estimation du temps d’utilisation

- Comparaison entre plusieurs profils utilisateurs
